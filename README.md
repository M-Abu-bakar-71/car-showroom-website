🚗 Car Showroom Management System (CSMS)

A web-based Car Showroom Management System designed to manage vehicle inventory, customer interactions, and order processing through a structured and role-based platform. This project is built with a strong focus on database design, system organization, and academic best practices.

📖 Overview

The Car Showroom Management System (CSMS) automates the core operations of a car showroom by providing separate access for Customers and Administrators. The system eliminates manual record handling and ensures accurate data storage, easy management, and a smooth user experience.

👥 User Roles
Customer

Register and authenticate securely

Browse showroom pages (Home, About, Cars)

View cars brand-wise

Add vehicles to cart

Checkout and place orders

Admin

Secure admin authentication

Perform CRUD operations on cars

Manage inventory records

View and manage customer orders

Maintain overall system data

✨ Features

Role-based authentication system
